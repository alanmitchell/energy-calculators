## Application Description

This is a Python web application that provides a number of different calculators, each calculator residing on its own web page. Starting the app will bring you to the home page, which allows you to select the specific calculator to use. The home page should be laid out as a set of Cards, each card with a description of the calculator and a button or link to move to that particular calculator page.

## Architecture

* Python as the programming language.
* NiceGUI for the UI framework.
* Requests for API calls.
* Pandas for manipulation of Tabular data.
* Matplotlib for static graphs and diagrams.
* Plotly for interactive graphs.
* Use type hints for all variables and function parameters.
* Include docstrings for all functions, classes, and modules.
* Use uv (e.g. uv add) to manage dependencies and the virtual environment.
