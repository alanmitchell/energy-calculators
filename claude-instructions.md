I would like to build a Python web application that provides a number of different calculators, each calculator residing on its own web page. Starting the app will bring you to the home page, which allows you to select the specific calculator to use. The home page should be laid out as a set of Cards, each card with a description of the calculator and a button or link to move to that particular calculator page. I would like to use the NiceGUI framework for the application. To start with, can you build a simple Celsius to Fahrenheit converter as one calculator. For a second calculator can you allow the conversion of a dry-bulb temperature plus relative humidity value into a dewpoint temperature, a wet-bulb temperature, and an absolute humidity value.
